## 收获与体会
1. 对于一项知识（编程语言/框架）的学习，**必须要做到理论和实践相结合**。

   就我目前而言，方法论如下：
   - 官方文档很重要，但是由于大部分都是英文，所以阅读起来很困难，这是一件目前无法解决的问题，英语的学习只能慢慢来。
   - 实例非常重要，可以找官方文档的实例，也可以找一些优秀博主的教程。
   - 对于找到的优秀教程，一定要按照教程的操作自己动手做一下，只有这样才能收获更多！这一点非常重要！
